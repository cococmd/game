# Coco's Hungry Snake 🐍✨

水晶画质的贪吃蛇 · 随机水果 🍎🍌🍐

## 手机直接玩（GitHub Pages）

开启 Pages 后，用手机浏览器打开：

**https://cococmd.github.io/game/**

### 第一次开启 Pages（只需做一次）

1. 打开 [github.com/cococmd/game](https://github.com/cococmd/game)
2. 点 **Settings** → 左侧 **Pages**
3. **Build and deployment** → Source 选 **Deploy from a branch**
4. Branch 选 **main**，文件夹选 **/ (root)**，点 **Save**
5. 等 1～2 分钟，刷新 Pages 页面，出现绿色网址即可

### 用 GitHub Desktop 更新游戏

1. 打开 GitHub Desktop，选中 **game** 仓库
2. 左侧会看到改动的文件，填写说明后点 **Commit to main**
3. 点 **Push origin**
4. 等约 1 分钟，手机打开上面的网址即可玩到最新版

## 电脑本地玩

双击 `index.html`，或在终端：

```bash
cd ~/Documents/cocodev/game
python3 -m http.server 8888
```

浏览器打开 `http://localhost:8888`

## 操作

| 平台 | 操作 |
|------|------|
| 手机 | 滑动游戏区 或 点击下方方向键 |
| 电脑 | 方向键 / WASD · 空格暂停 |
